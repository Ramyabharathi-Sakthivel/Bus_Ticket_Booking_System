PROJECT OVERVIEW :

The Bus Ticket Booking System is a Java-based application designed to simplify the process of reserving bus tickets.
It allows users to view available buses, select routes, and book or cancel tickets efficiently.
The system focuses on providing a basic and user-friendly interface for managing passenger details and ticket records, making it suitable for learning core Java concepts and application flow.

FEATURES :

View available bus routes and schedules
Book bus tickets with passenger details
Cancel booked tickets
Display booking information and seat availability
Basic record management for buses and passengers

TECHNOLOGIES USED :

Java – Core programming language
OOP Concepts – Classes, Objects, Inheritance, Encapsulation
File Handling / Console I/O – For storing and retrieving booking data (if implemented)
IDE Support – Eclipse
